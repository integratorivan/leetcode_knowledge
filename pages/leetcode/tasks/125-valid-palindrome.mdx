[Valid palindrome](https://leetcode.com/problems/valid-palindrome/description/)

```javascript
var isPalindrome = function(s) {
    const string = s.toLowerCase().replace(/[^a-z0-9]/g, '');
    let left = 0;
    let right = string.length - 1;
  
    while (left < right) {
      if (string[left] !== string[right]) {
        return false;
      }
      left++;
      right--;
    }
  
    return true;
};
```

1. Строка `s` преобразуется в нижний регистр с помощью метода `toLowerCase()`.
2.	Из полученной строки удалются все символы, не являющиеся буквенно-цифровыми, с использованием регулярного выражения `/[^a-z0-9]/g` и метода `replace()`. Это делается для того, чтобы игнорировать пробелы, знаки препинания и другие символы при проверке на палиндром.
3.	Затем используется два указателя `left` (указывающий на первый символ строки) и `right` (указывающий на последний символ строки).
4.	В цикле `while` происходит сравнение символов по обоим концам строки. Если символы не совпадают, то функция возвращает `false`, означая, что строка не является палиндромом.
5.	Если весь цикл выполнен и ни одно несовпадение символов не обнаружено, то функция возвращает `true`, указывая на то, что строка является палиндромом.

- Сложность данного решения составляет O(n), где "n" - это длина входной строки (s).
